🚀 Excited to share my second latest project — GRC Compliance Assistant for KSA Regulations!

As a GRC professional, I've always wanted a smarter way to navigate complex KSA regulatory frameworks. So I built one — and then made it agentic. 🛡️🤖

I developed an AI-powered compliance tool that covers THREE major Saudi frameworks in a single app, with two modes of intelligence:

📋 Frameworks covered:
- 🏦 SAMA Cyber Security Framework (CSF)
- 🔒 Personal Data Protection Law (PDPL)
- 🛡️ NCA Essential Cybersecurity Controls (ECC)

💬 Mode 1 — Chat Mode (RAG Chatbot):
Ask a specific compliance question → the app retrieves the most relevant chunks from the official PDF → Claude answers with source page references. Fast, grounded, zero hallucination.

🤖 Mode 2 — Agent Mode (Agentic AI):
Give the agent a complex compliance goal → Claude autonomously decides which frameworks to search, runs multiple searches, evaluates the results, and synthesises a comprehensive answer — all without you directing each step.

Example agent goals:
→ "Compare incident response requirements across SAMA CSF, PDPL, and NCA ECC"
→ "What do I need to reach SAMA CSF maturity level 3 in cybersecurity governance?"
→ "List all penalties under PDPL and what triggers them"

🔧 How I built it:
- Claude Tool Use (Anthropic) for the agentic loop — Claude calls search tools autonomously
- RAG pipeline using LangChain + ChromaDB for grounded retrieval
- HuggingFace sentence-transformers for free local embeddings
- Framework metadata tagging for smart cross-framework filtering
- Follow-up question suggestions after every answer
- One-click copy button for pasting answers into audit reports
- Fully deployed on Streamlit Community Cloud

💡 What it does:
Select a framework, ask a question or give a complex goal, and get a structured compliance answer — with source pages, maturity levels, legal obligations, and implementation guidance, all in seconds.

🎯 Built for:
GRC professionals, cybersecurity auditors, compliance officers, DPOs, and anyone working within Saudi Arabia's regulatory landscape — banking, fintech, government, or any regulated entity.
