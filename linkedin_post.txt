🚀 Excited to share my latest project — GRC Compliance Assistant for Saudi Regulations!

As a GRC professional, I've always wanted a smarter way to navigate complex Saudi regulatory frameworks. So I built one. 🛡️

I developed an AI-powered compliance chatbot that now covers THREE major Saudi frameworks — all in a single app, with zero hallucination.

📋 Frameworks covered:
- 🏦 SAMA Cyber Security Framework (CSF)
- 🔒 Personal Data Protection Law (PDPL)
- 🛡️ NCA Essential Cybersecurity Controls (ECC)

🔧 How I built it:
- RAG (Retrieval Augmented Generation) pipeline using LangChain + ChromaDB
- Anthropic Claude AI for structured, professional compliance answers
- Free local embeddings via HuggingFace sentence-transformers (no API cost)
- Single ChromaDB collection with framework metadata tagging for smart filtering
- Dynamic UI — suggested questions and domain filters change per selected framework
- Fully deployed on Streamlit Community Cloud — publicly accessible
- Code published on GitHub

💡 What it does:
Select a framework, ask any compliance question, and get an instant structured answer — with the exact PDF page it came from. Maturity levels, control objectives, legal obligations, penalties, and implementation guidance, all in seconds.

🎯 Built for:
GRC professionals, cybersecurity auditors, compliance officers, DPOs, and anyone working within Saudi Arabia's regulatory landscape — banking, fintech, government, or any regulated entity.
